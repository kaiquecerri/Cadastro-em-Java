# 🎨 ConnectArt (Demonstração em Java)

## 📌 Sobre

Este projeto é **apenas uma demonstração simples** em Java para representar a lógica básica de cadastro de usuários no sistema ConnectArt.

Ele não representa o sistema final, sendo utilizado apenas para fins acadêmicos e de aprendizado.

---

## 🎯 Objetivo

Demonstrar conceitos básicos de:

* Entrada de dados (`Scanner`)
* Estruturas de repetição (`while`, `do-while`)
* Condições (`if`, `switch`)
* Manipulação de arrays
* Validação de dados (usuário e senha)

---

## ⚙️ Funcionalidades simuladas

* Cadastro de usuários (Professor ou Aluno)
* Validação de:

  * Nome de usuário único
  * Tamanho mínimo de usuário (3 caracteres)
  * Senha com mínimo de 8 caracteres
  * Confirmação de senha
* Exibição dos usuários cadastrados

---

## ⚠️ Observações importantes

* Os dados são armazenados apenas em memória (array)
* Não há banco de dados
* Não há criptografia de senha
* Não possui sistema de login
* Interface apenas em terminal

👉 Ou seja: **não é um sistema real, apenas uma simulação de lógica**

---

